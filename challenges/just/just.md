# Just Command Runner -- Challenges & Exercises

> 50 hands-on exercises for the `just` command runner, organized across 5 difficulty levels.
> Each exercise follows a progressive learning path from basic recipe syntax to production-grade orchestration.

**Requirements**: `just` installed (`brew install just` / `cargo install just`), terminal with `bash` available.

**Convention**: Each exercise uses a clean directory with a `justfile`. Run `just` with no arguments to list available recipes.
