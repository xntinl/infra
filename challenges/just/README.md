# Just Challenges & Exercises

> 19 ejercicios practicos de `just` command runner organizados en 4 niveles.
> Cada ejercicio incluye una guia completa con conceptos clave, codigo, verificacion y recursos.

**Requisitos**: `just` instalado (`brew install just` / `cargo install just`), terminal con bash disponible.

**Convencion**: Cada ejercicio usa un directorio limpio con un `justfile`. Ejecutar `just` sin argumentos para ver las recetas disponibles.
