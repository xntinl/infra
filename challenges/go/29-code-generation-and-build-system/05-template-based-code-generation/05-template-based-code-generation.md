# Exercise 29.5: Template-Based Code Generation

## Prerequisites

Before starting this exercise, you should be comfortable with:

- AST parsing with `go/ast` and `go/parser` (Exercise 29.4)
- Writing custom code generators (Exercise 29.3)
- Go `text/template` package

## Learning Objectives

By the end of this exercise, you will be able to:

1. Combine AST parsing with template-based output for fully automatic code generation
2. Extract struct metadata (fields, types, tags) from source and feed it into templates
3. Generate multiple output files from a single generator run
4. Build a practical generator that produces constructor and builder pattern code

## Why This Matters

The most powerful Go generators combine AST analysis with template rendering. This pattern is used by real-world tools like `protoc-gen-go`, `ent`, and `sqlc`. By mastering this combination, you can build generators tailored to your domain that eliminate thousands of lines of hand-written boilerplate.

---

## Problem

Build a generator that reads Go struct definitions from a source file and produces:

1. A `New<Struct>` constructor function with all exported fields as parameters
2. A builder pattern with a fluent `With<Field>` method for each exported field
3. A `Validate()` method that checks required fields (those without `omitempty` in their JSON tag)

### Hints

- Parse the source file with `go/parser` to discover structs and their fields
- Extract JSON tag names using `reflect.StructTag` to parse raw tag strings
- Use separate templates for constructors and builders
- Write output through `go/format.Source` to ensure valid formatting

### Step 3: Write the generator

Create `cmd/structgen/main.go`:

```go
//go:build ignore

package main

import (
	"bytes"
	"flag"
	"fmt"
	"go/ast"
	"go/format"
	"go/parser"
	"go/token"
	"os"
	"reflect"
	"strings"
	"text/template"
)

type Field struct {
	Name     string
	Type     string
	JSONName string
	Required bool
}

type StructInfo struct {
	Name   string
	Fields []Field
}

var tmpl = template.Must(template.New("gen").Funcs(template.FuncMap{
	"lower": strings.ToLower,
}).Parse(`// Code generated by structgen; DO NOT EDIT.

package {{ .Package }}

import "fmt"

{{ range .Structs }}
// New{{ .Name }} creates a new {{ .Name }} with required fields.
func New{{ .Name }}({{ range $i, $f := .Fields }}{{ if $f.Required }}{{ if $i }}, {{ end }}{{ lower $f.Name }} {{ $f.Type }}{{ end }}{{ end }}) *{{ .Name }} {
	return &{{ .Name }}{
		{{- range .Fields }}{{ if .Required }}
		{{ .Name }}: {{ lower .Name }},
		{{- end }}{{ end }}
	}
}

// {{ .Name }}Builder provides a fluent API to build {{ .Name }}.
type {{ .Name }}Builder struct {
	v {{ .Name }}
}

// Build{{ .Name }} starts building a {{ .Name }}.
func Build{{ .Name }}() *{{ .Name }}Builder {
	return &{{ .Name }}Builder{}
}

{{ range .Fields }}
// With{{ .Name }} sets the {{ .Name }} field.
func (b *{{ $.Name }}Builder) With{{ .Name }}(val {{ .Type }}) *{{ $.Name }}Builder {
	b.v.{{ .Name }} = val
	return b
}
{{ end }}

// Build returns the constructed {{ .Name }}.
func (b *{{ .Name }}Builder) Build() *{{ .Name }} {
	result := b.v
	return &result
}

// Validate checks that all required fields are set.
func (c *{{ .Name }}) Validate() error {
	var missing []string
	{{- range .Fields }}{{ if .Required }}
	{{ if eq .Type "string" -}}
	if c.{{ .Name }} == "" {
		missing = append(missing, "{{ .JSONName }}")
	}
	{{ else if eq .Type "int" -}}
	if c.{{ .Name }} == 0 {
		missing = append(missing, "{{ .JSONName }}")
	}
	{{ end -}}
	{{ end }}{{ end -}}
	if len(missing) > 0 {
		return fmt.Errorf("missing required fields: %v", missing)
	}
	return nil
}
{{ end }}
`))

type templateData struct {
	Package string
	Structs []StructInfo
}

func main() {
	input := flag.String("input", "", "input Go source file")
	output := flag.String("output", "", "output file")
	flag.Parse()

	if *input == "" || *output == "" {
		fmt.Fprintln(os.Stderr, "usage: structgen -input=file.go -output=out.go")
		os.Exit(1)
	}

	fset := token.NewFileSet()
	file, err := parser.ParseFile(fset, *input, nil, parser.ParseComments)
	if err != nil {
		fmt.Fprintf(os.Stderr, "parse error: %v\n", err)
		os.Exit(1)
	}

	var structs []StructInfo

	ast.Inspect(file, func(n ast.Node) bool {
		ts, ok := n.(*ast.TypeSpec)
		if !ok {
			return true
		}
		st, ok := ts.Type.(*ast.StructType)
		if !ok {
			return true
		}

		info := StructInfo{Name: ts.Name.Name}
		for _, field := range st.Fields.List {
			if len(field.Names) == 0 || !ast.IsExported(field.Names[0].Name) {
				continue
			}

			f := Field{
				Name:     field.Names[0].Name,
				Type:     exprString(field.Type),
				Required: true,
			}

			if field.Tag != nil {
				raw := strings.Trim(field.Tag.Value, "`")
				tag := reflect.StructTag(raw)
				jsonTag := tag.Get("json")
				parts := strings.Split(jsonTag, ",")
				f.JSONName = parts[0]
				for _, opt := range parts[1:] {
					if opt == "omitempty" {
						f.Required = false
					}
				}
			}

			info.Fields = append(info.Fields, f)
		}

		structs = append(structs, info)
		return true
	})

	data := templateData{
		Package: file.Name.Name,
		Structs: structs,
	}

	var buf bytes.Buffer
	if err := tmpl.Execute(&buf, data); err != nil {
		fmt.Fprintf(os.Stderr, "template error: %v\n", err)
		os.Exit(1)
	}

	formatted, err := format.Source(buf.Bytes())
	if err != nil {
		fmt.Fprintf(os.Stderr, "format error: %v\n%s\n", err, buf.String())
		os.Exit(1)
	}

	if err := os.WriteFile(*output, formatted, 0644); err != nil {
		fmt.Fprintf(os.Stderr, "write error: %v\n", err)
		os.Exit(1)
	}

	fmt.Printf("Generated %s with %d struct(s)\n", *output, len(structs))
}

func exprString(expr ast.Expr) string {
	switch e := expr.(type) {
	case *ast.Ident:
		return e.Name
	case *ast.SelectorExpr:
		return exprString(e.X) + "." + e.Sel.Name
	case *ast.StarExpr:
		return "*" + exprString(e.X)
	case *ast.ArrayType:
		return "[]" + exprString(e.Elt)
	default:
		return fmt.Sprintf("%T", expr)
	}
}
```

### Step 4: Create main.go to demonstrate the generated code

Create `main.go`:

```go
package main

import "fmt"

func main() {
	// Using the constructor
	cfg := NewConfig("localhost", 5432, "mydb")
	fmt.Printf("Config: %+v\n", cfg)
	fmt.Println("Valid:", cfg.Validate())

	// Using the builder
	ep := BuildEndpoint().
		WithPath("/api/users").
		WithMethod("GET").
		WithAuth(true).
		Build()
	fmt.Printf("Endpoint: %+v\n", ep)
	fmt.Println("Valid:", ep.Validate())

	// Validation error
	empty := &Config{}
	fmt.Println("Empty valid:", empty.Validate())
}
```

### Step 5: Generate and run

```bash
go generate ./...
go run .
```
