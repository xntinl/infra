# infra

Cloud infrastructure learning platform with hands-on exercises covering AWS, Terraform, Kubernetes, and OPA.

## Structure

- **challenges/** — guided exercises organized by topic and difficulty (basic → insane)
  - **aws/** — networking, compute, storage, security, serverless, CI/CD, observability, and more
  - **terraform/** — HCL patterns, state management, module design, testing
  - **kubernetes/** — workloads, networking, security, observability
  - **opa/** — policy as code
  - **aws-saa-c03-solutions-architect/** — SA Associate exam prep
  - **aws-dva-c02-developer-associate/** — Developer Associate exam prep
- **platform/** — shared Terraform infrastructure
- **cost-guard/** — AWS cost monitoring and alerting
- **cloud-sweep/** — automated cloud resource cleanup

## Getting Started

Each exercise includes learning objectives, complete HCL code or TODO skeletons, best practices, verification commands, and cleanup instructions.
